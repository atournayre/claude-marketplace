# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Versioning Sémantique](https://semver.org/lang/fr/).

## [1.0.0] - 2025-11-15

### Added
- Commande `/output-styles:style-bullet-points` - Format listes à puces hiérarchiques
- Commande `/output-styles:style-genui` - UI générative avec styling moderne
- Commande `/output-styles:style-html-structured` - HTML sémantique propre
- Commande `/output-styles:style-markdown-focused` - Markdown complet pour lisibilité maximale
- Commande `/output-styles:style-table-based` - Tables Markdown pour organisation
- Commande `/output-styles:style-ultra-concise` - Minimal words maximum speed
- Commande `/output-styles:style-yaml-structured` - YAML structuré hiérarchique
